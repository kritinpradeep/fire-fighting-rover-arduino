# Fire Fighting Rover using Arduino

## 📌 Overview
This project is an autonomous fire fighting rover designed to detect and extinguish fires using multiple sensors and real-time processing.

## 🚀 Features
- Fire detection using flame sensors  
- Gas and temperature monitoring  
- Obstacle detection using ultrasonic sensors  
- Autonomous navigation  
- Automatic water pump activation for fire suppression  

## 🛠️ Tech Stack
- Arduino Uno  
- Embedded C / Arduino IDE  
- Sensors: Flame, Gas (MQ2), Temperature (LM35), Ultrasonic  
- Hardware: Motors, Relay Module, Water Pump  

## ⚙️ Working
The rover continuously monitors environmental conditions using sensors. Upon detecting fire, it navigates towards the source and activates the water pump to extinguish it.


## ▶️ How to Run
1. Upload the code to Arduino  
2. Assemble the hardware components  
3. Power the rover and test
